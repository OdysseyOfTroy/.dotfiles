# Scripts
| Script | Description |
| :----- | :---------- |
| `install.sh`| This script installs Homebrew and all Casks defined in `Brewfile`|
| `configure.sh`| Configures tools and applies dotfiles. | 
| `uninstall.sh` | Removes all packages, tools and config files. |
